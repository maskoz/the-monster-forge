# The Monster Forge

A Foundry VTT module — NPC generator for Shadowdark RPG (v12–v13, Shadowdark system v3.6.2+).

## Project Structure

- `module.json` — module manifest (version, packs, compatibility)
- `packs/action-monsters` — RollTable compendium (15 tables, 300 entries)
- `packs/monster-forge-macros` — Macro compendium (The Monster Forge macro)
- `CHANGELOG.md` — version history

## Workflow Notes

- Do not use git worktrees on this project.
- Do not create pull requests. Commit directly to main.
